How to Use

1. Generate an image in **Lovart**.
2. Press **F12** to open Browser Developer Tools and navigate to the **Network** tab.
3. Click **"Edit elements"** in Lovart. This will initiate the image layering process.
4. Several data packets starting with `task?task_id` will appear in the Network tab. Wait for the layering to complete and locate the **last** correct data packet (usually, the correct packet is followed by the image data packets).
5. Right-click the correct data packet and select **Copy** -> **Copy response**.
6. Open the **Conversion Tool** and paste the copied code. The converted layers will be displayed below automatically. Once confirmed, click the **Download** button to obtain the PSD file.
